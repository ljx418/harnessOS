# Shared Imag2 Style

Use this shared style for every HarnessOS project-introduction image.

```text
Create a polished Chinese SaaS product infographic for HarnessOS, a governed
multi-agent workflow operating system. Light theme, dense but readable,
professional AI workflow product UI, calm semantic colors: blue for selected,
green for accepted evidence, amber for limited or waiting states, red for
blocked No-Go boundaries, neutral gray for infrastructure. Use clean panels,
workflow canvas fragments, agent node cards, curved connectors, inspector
panels, evidence tags and small architecture blocks. Avoid copied brand assets,
fake logos, distorted Chinese body text, fake metrics, exaggerated gradients,
dark sci-fi style, and any claim that the system is production-ready or fully
complete. Use only large short Chinese labels where text is needed; detailed
copy will be rendered in HTML outside the image.
```
