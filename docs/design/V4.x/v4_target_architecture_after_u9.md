## 2. Target Architecture

V4 的目标架构是 dev/local Headless-first workflow operating layer：

```text
User Intent
  -> Mission Console
  -> Interaction Orchestrator
  -> WorkflowSpec / Diff
  -> Workflow Blueprint
  -> User Confirmation
  -> Headless Workflow Core
  -> Runtime Report
  -> Review Console
  -> Evidence Chain
```

核心组件：

| Component | Role | V4-U9 Status | Runtime Truth |
| --- | --- | --- | --- |
| Headless Workflow Core | dev/local workflow execution and artifact production | supported for scoped scenarios | yes |
| Mission Console | natural language / command entry, intent capture, proposal handoff | dev/local baseline | no |
| Interaction Orchestrator | shared intent, action, and handoff coordination | read-model contract | no |
| Experience State Machine | shared UX state projection across heads | read-model contract | no |
| Agent Policy Layer | prevents source=agent from direct durable mutation | guard baseline | no |
| Runtime Capability Matrix | supported / partial / planned / unsupported boundary | audit read model | no |
| WorkflowSpec Registry | spec hash and linked runtime refs | read model | no |
| Workflow Blueprint | drawio / structural visualization | read-only report | no |
| Runtime Report | station, artifact, quality, attempt state report | read-only report | no |
| Review Console | failure explanation, repair proposal, user-confirmed handoff | governed handoff | no |
| Evidence Chain | proposal, handoff, user_confirmed, policy, runtime result audit | read-only report | no |

## 3. Runtime Truth Boundary

V4-U9 keeps runtime truth limited to governed workflow/runtime objects and evidence generated by the scoped dev/local paths.

The following are not runtime truth:

```text
WorkflowSpec
WorkflowSpec Registry
Drawio
HTML Report
Mission Console transcript
Review Console explanation
Evidence Chain HTML view
EventBridge payload
```

Durable mutation remains governed:

```text
user_confirmed=true required
source=agent cannot execute durable mutation
EventBridge only triggers refresh
Evidence Chain is read-only
Drawio is visualization only
HTML Report is read-only
```

## 4. Dev/Local Capability Boundary

V4-U9 proves scoped dev/local capabilities:

1. Local recursive Markdown summary workflow.
2. Real provider-backed local technical document summary.
3. Mission Console to WorkflowSpec / Diff / Blueprint evidence.
4. Runtime Report, Artifact, Quality, Review Console, and Evidence Chain evidence.
5. Provider-backed dev/local serial video, parallel deliberation, and engineering workflow scenario evidence.
6. Final human acceptance and V5 handoff package.

V4-U9 does not prove:

1. Production auth.
2. Production tenant isolation.
3. Production token or credential lifecycle.
4. Production observability or audit export.
5. Production external app onboarding.
6. Distributed multi-agent runtime.
7. Full Web Studio productization.
8. Agent executor or production controlled executor readiness.
