## Verified Behavior

- Agent-created folder summary workflow is represented as proposal state.
- Canvas displays pending ghost nodes for the 9 logical V4.1-A nodes.
- Folder read requires explicit authorization.
- Debug scan returns tree and counts but does not generate summaries or artifacts.
- Apply, publish, and run require `user_confirmed=true` and reject `source=agent`.
- Runner generates:
  - `AgentOS_总结.md`
  - `前端低代码_总结.md`
  - `项目复盘_总结.md`
  - `总览总结.md`
  - `quality_report.json`
- Quality report records `未支持/test.pdf` and `空文件夹`.
- Stored workflow instance and artifacts can be read back through BFF routes.
- Browser-facing client uses `/bff/v4_1/*` routes and does not call `/v1/rpc` or `/v1/events/subscribe`.
- Raw Markdown source content is not returned by debug scan, error response, or authorization DTO.
