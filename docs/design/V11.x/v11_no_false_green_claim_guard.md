# V11 No False Green Claim Guard

## Allowed Claims

- V11 planning complete: real-time explainable Mission TUI development package ready for review.
- V11-1 complete: real-time TUI event stream slice ready for review.
- V11-2 complete: Mission TUI command interaction slice ready for review.
- V11 complete: real-time explainable Mission TUI interaction baseline ready for review.

## Forbidden Positive Claims

- production ready
- full production GA
- Agent executor ready
- complete Workflow Studio ready
- full multi-Agent orchestration ready
- autonomous workflow editing ready
- unrestricted terminal worker ready
- production terminal automation ready
- Codex/Claude Code parity complete
- 生产可用
- 全面生产可用
- Agent 执行器已完成
- 完整工作流工作台已完成
- 多 Agent 编排已完全完成
- 自主工作流编辑已完成
- 终端自动化生产可用
- 已完全追平 Codex
- 已完全追平 Claude Code

## Safe Contexts

Forbidden phrases may appear only in:

- this No False Green document,
- stop conditions,
- forbidden interpretations,
- audit questions,
- drawio warning boxes,
- boundary explanations.

They must not appear as release notes, completion claims, dashboard status or
positive acceptance result.

## Stage Claim Order Rule

V11-N complete cannot be claimed unless every earlier V11 stage is PASS or has
an explicitly accepted bounded non-blocking exception with evidence. A later
stage evidence package cannot retroactively convert an earlier missing, failed
or blocked stage into PASS.
