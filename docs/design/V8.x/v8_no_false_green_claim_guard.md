## Allowed Ready-For-Review Claims

```text
V8 complete: station-agent workflow pilot ready for review.
```

The phrase `ready for review` must not be shortened to `ready` in summaries or completion notes.

## Redaction Terms

Evidence, HTML reports, JSON artifacts and logs must not contain:

```text
raw prompt
raw file content
raw provider payload
raw connector payload
raw artifact content
API key
Bearer token
signed URL
raw secret
capability_token
subscription_token
```
