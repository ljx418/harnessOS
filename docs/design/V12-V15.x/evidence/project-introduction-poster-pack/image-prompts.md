# Image Prompts

These prompts can be reused in imag2, Gemini, Minimax or another image model to
regenerate supporting poster illustrations. The generated images are supporting
marketing/design assets only.

## Shared Style

```text
Modern Chinese SaaS product poster, clean light theme, professional AI workflow
operating system, low-code workflow canvas, agent node cards, curved connection
lines, right inspector panel, bottom evidence workbench, calm blue/green/amber
semantic color system, crisp line icons, dense but readable enterprise UI,
high-resolution editorial composition, no copied brand assets, no Xpert logo,
no fake production-ready claim, no distorted Chinese text, no emoji icons.
```

## Master Poster

```text
Create a 16:9 hero poster for HarnessOS, a multi-Agent workflow operating
system. Center: a polished low-code canvas with several Agent nodes connected by
curved edges. Left: product navigation rail and workflow list. Right: inspector
showing role, goal, tools, quality and evidence refs. Bottom: chat and evidence
tabs. Overlay a clean Chinese headline area: HarnessOS, 把自然语言变成可审查的多
Agent 工作流. Show five small process chips: 输入目标, 生成草案, 查看分工, 审查证据,
用户确认. Use refined blue, green, amber and neutral palette.
```

## Quickstart Poster

```text
Create a Chinese infographic poster showing a five-step user journey for an AI
workflow product. Use five connected panels: input a natural-language goal,
generate workflow draft, inspect Agent stations, review outputs and evidence,
confirm next step. Include subtle browser workbench UI fragments, timeline and
evidence chain visuals. Clean enterprise SaaS style, no exaggerated gradients.
```

## Workflow Map Poster

```text
Create a workflow map poster for an AI workbench. Show five workflow families as
cards connected to a central HarnessOS canvas: multi-Agent debate, video
storyboard creation, local document summary, coding proposal workflow, evidence
audit. Each card should have small abstract UI details and evidence refs. Use a
professional product architecture poster style.
```

## Use Cases Poster

```text
Create a polished Chinese SaaS use-case poster for a small AI studio. Show five
teams around one workflow workbench: creators, developers, knowledge workers,
product managers and auditors. Each team card points to a specific workflow
artifact: storyboard, diff proposal, document summary, WorkflowSpec, evidence
chain. Use bright but restrained product colors.
```

## Business Outcomes Poster

```text
Create a business outcomes poster for HarnessOS. Visualize reusable workflows,
audit cost reduction, standardized Agent stations, visible blocked states, and
handoff-ready evidence assets. Use product UI metrics cards, but avoid fake
numeric claims. Make it look like a credible enterprise product one-pager.
```

## Trust And Evidence Poster

```text
Create a trust and evidence poster for an AI workflow platform. Show a node
selected on canvas and a magnified evidence chain: Agent role, input/output refs,
quality gate, runtime report, redaction status, human confirmation. Use green
for passed evidence, amber for waiting confirmation, red for blocked, blue for
selected. High clarity, Chinese product UI poster.
```
