# HarnessOS Project Introduction Poster Pack

This folder contains a user-facing poster set for introducing HarnessOS
application scenarios, quick-start flow, workflow families and expected business
outcomes.

## Files

| File | Purpose |
| --- | --- |
| `index.html` | Review page and poster rendering source. |
| `poster-master.png` | Main one-image project introduction poster. |
| `poster-01-quickstart.png` | Quick-start poster. |
| `poster-02-workflow-map.png` | Workflow family map. |
| `poster-03-use-cases.png` | User scenario poster. |
| `poster-04-business-outcomes.png` | Business outcome poster. |
| `poster-05-trust-and-evidence.png` | Evidence and trust poster. |
| `poster-copy.md` | Copywriting source. |
| `image-prompts.md` | Reusable image generation prompts. |
| `review-checklist.json` | Review checklist and claim boundary. |

## Boundary

This is a design and communication package. It is not runtime evidence, BFF
evidence, browser implementation evidence, Xpert parity evidence, complete
Workflow Studio evidence or production readiness evidence.

Allowed statement:

```text
HarnessOS project introduction poster set ready for review.
```
