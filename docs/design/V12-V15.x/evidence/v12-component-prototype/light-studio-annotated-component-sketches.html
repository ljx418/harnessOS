总图验收重点

你应该能一眼看出工作台由 6 块组成，每块承担不同职责；Chat、画布和 Inspector 之间有明确关系。

V12 边界

画布是 read-only foundation；Chat 只产生 proposal；运行、发布、写入都必须等待确认和 BFF/DTO 证据。

下一步

如果总图方向成立，V12-0P 应把它转为 Figma 或高保真 HTML 原型，并补足焦点、hover、空态、错误态和响应式。
