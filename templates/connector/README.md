# Connector Template

This directory is a V3.5-G scaffold for creating new connector descriptors. It
is not a runtime connector instance and must not be auto-discovered by
`connector.health`.

To instantiate a connector manually:

1. Copy this directory to `connectors/<new_connector>` or to a test fixture.
2. Edit `descriptor.json`.
3. Pass the descriptor path explicitly to a test or registry configuration.

Discovery only reads `descriptor.json`. Example `health.py` and `tools.py` files
are development notes and must not be imported or executed by descriptor
discovery.

This phase does not provide a `harnessos connector init` CLI.

Compatibility warnings are generated by the loader or health result; do not
declare `compatibility_warnings` as descriptor truth.
