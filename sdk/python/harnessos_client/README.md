# harnessos_client

Python SDK MVP for harnessOS V3.5-D.

This package is a thin JSON-RPC protocol client. Runtime code must not import
harnessOS server internals such as GatewayService, RuntimeAdapter, Core Store,
or server schema registries.

V3.5-D status:

- usable for local/backend integration smoke
- not production-ready
- not external app ready
- not V3.5-MVP complete
